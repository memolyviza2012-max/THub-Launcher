# 🛠️ คู่มือการสร้างฟอนต์ภาษาไทยสำหรับ XCOM ด้วย UDK

การสร้างฟอนต์ Canvas ของ Unreal Engine 3 ด้วยตัวเอง เป็นสุดยอดทักษะของนักม็อดเลยครับ! และนี่คือขั้นตอนโดยละเอียดตั้งแต่ต้นจนจบ:

## 1. การหาดาวน์โหลดโปรแกรม UDK
**UDK (Unreal Development Kit)** คือเอนจิ้นสร้างเกมเวอร์ชั่นฟรีที่ Epic Games เคยปล่อยออกมา เนื่องจาก XCOM ออกวางจำหน่ายในปี 2012 เราควรใช้ UDK รุ่นปีใกล้เคียงกันครับ
- แนะนำให้ไปดาวน์โหลดที่เว็บ **ModDB** (ค้นหาใน Google ว่า `UDK 2012 Download ModDB` หรือ `UDK 2011`)
- เมื่อโหลดเสร็จให้ติดตั้งลงคอมพิวเตอร์ของคุณตามปกติ

## 2. การเตรียมตัวอักษรที่จะฝังลงฟอนต์
ระบบของ UE3 จะทำการ "ถ่ายภาพ" (Rasterize) ตัวอักษรที่เรากำหนดเก็บไว้เป็นไฟล์ภาพ ดังนั้นเราต้องเตรียมตัวอักษรทั้งหมดให้มันครับ ให้คุณก๊อปปี้ข้อความในกล่องด้านล่างนี้เตรียมไว้:

```text
 !"#$%&'()*+,-./0123456789:;<=>?@ABCDEFGHIJKLMNOPQRSTUVWXYZ[\]^_`abcdefghijklmnopqrstuvwxyz{|}~กขฃคฅฆงจฉชซฌญฎฏฐฑฒณดตถทธนบปผฝพฟภมยรฤลฦวศษสหฬอฮฯะัาำิีึืฺุู฿เแโใไๅๆ็่้๊๋์ํ๎๏๐๑๒๓๔๕๖๗๘๙๚๛
```

## 3. ขั้นตอนการสร้างฟอนต์ใน UDK
1. เปิดโปรแกรม **UDK Editor** ขึ้นมา (อาจจะใช้เวลาโหลดครั้งแรกสักพัก)
2. เมื่อเปิดมาแล้ว ให้หาหน้าต่าง **Content Browser** (มักจะอยู่ด้านล่าง หรือกดไอคอนรูปโฟลเดอร์แว่นขยายด้านบน)
3. ที่มุมล่างซ้ายของ Content Browser ให้คลิกปุ่ม **New** (สร้างใหม่)
4. จะมีหน้าต่างขึ้นมาให้ตั้งค่า ดังนี้:
   - **Package:** พิมพ์ชื่อไฟล์ที่ต้องการ เช่น `XcomThaiFont` (ห้ามเว้นวรรค)
   - **Grouping:** ปล่อยว่างไว้
   - **Name:** พิมพ์ชื่อฟอนต์ เช่น `SubtitleThai` (ห้ามเว้นวรรค)
   - **Factory:** เลือกเป็น `Font`
   - กดปุ่ม **OK**
5. จะมีหน้าต่าง **Font Properties** เด้งขึ้นมา:
   - **FontName:** เลือกฟอนต์ภาษาไทยที่มีในเครื่องคอมของคุณ (เช่น `Tahoma`, `DB Helvethaica`, หรือ `Kanit` ที่อ่านง่ายๆ)
   - **Height:** ขนาดฟอนต์ แนะนำให้ลองตั้งสัก `24` หรือ `28`
   - **CharacterSet:** เปลี่ยนจาก Default เป็น `Custom`
   - **Chars:** ให้ก๊อปปี้ตัวอักษรอังกฤษและไทยจากข้อ 2. มา **วาง (Paste)** ลงในช่องนี้ให้หมด
   - กดปุ่ม **OK**
6. โปรแกรมจะทำการประมวลผลแป๊บเดียว แล้วคุณจะเห็นไอคอนฟอนต์ของคุณปรากฏใน Content Browser (ถ้าดับเบิลคลิกเข้าไปดู จะเห็นว่าโปรแกรมสร้างภาพตัวอักษรไทยเตรียมไว้ให้เรียบร้อยแล้ว!)

## 4. การเซฟไฟล์
1. ในหน้า Content Browser ให้คลิกขวาที่ชื่อแพ็คเกจ `XcomThaiFont` (ฝั่งซ้ายมือ) แล้วเลือก **Save**
2. เลือกตำแหน่งเซฟไฟล์ไว้ที่ไหนก็ได้ คุณก็จะได้ไฟล์ **`XcomThaiFont.upk`** มาครอบครองครับ! 🎉

## 5. การติดตั้งลงในเกม XCOM
1. นำไฟล์ `XcomThaiFont.upk` ไปวางในโฟลเดอร์ `F:\SteamLibrary\steamapps\common\XCom-Enemy-Unknown\XComGame\CookedPCConsole`
2. เข้าไปแก้ไขไฟล์ `DefaultEngine.ini` ของเกม
3. ค้นหาบรรทัด `SubtitleFontName=UEFonts.SubtitleFont`
4. เปลี่ยนให้ชี้มาที่ฟอนต์ของเราแบบนี้:
   ```ini
   SubtitleFontName=XcomThaiFont.SubtitleThai
   ```
5. เซฟไฟล์ INI แล้วเข้าเกมเพื่อทดสอบฉากคัตซีนได้เลยครับ!

> [!NOTE]
> **ข้อควรรู้เรื่องสระลอย:** 
> การใช้วิธีนำฟอนต์ภาษาไทยมาตรฐานเข้าไปตรงๆ อาจจะทำให้เกิดอาการ "สระลอย" หรือ "สระซ้อนทับกันไม่เป๊ะ" เล็กน้อย (เช่น คำว่า "ที่" ไม้เอกอาจจะลอยสูง) นี่เป็นข้อจำกัดของเอนจิ้นเกมยุคเก่าครับ 
> 
> แต่มันจะทำให้อ่านภาษาไทยออก **100%** โดยไม่ต้องเสียเวลาทำ PUA หรือไปฝังซับในวิดีโอเลยครับ ถือเป็นจุดเริ่มต้นที่คุ้มค่าที่สุด!
