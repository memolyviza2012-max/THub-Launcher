# 🚀 บทที่ 4: การใช้งานระบบ THub (Mastering THub Ecosystem)

**THub Launcher** ถูกสร้างขึ้นเพื่อลบความยุ่งยากในการแปลเกม ด้วยเครื่องมือ "Flagship Products" 4 ทหารเสือของเรา:

## 1. TStudio (เครื่องมือแปลสำหรับงานคราฟต์)
- **หน้าที่:** ใช้แปลไฟล์ `.json`, `.csv` แบบบรรทัดต่อบรรทัด
- **จุดเด่น:** มีระบบเช็กบริบท (Context) ช่วยให้การแปลไม่ออกทะเล และสามารถเช็กความยาวตัวอักษรว่าล้นกรอบเมนูหรือไม่ 
- **เหมาะสำหรับ:** การแปลเมนู UI หรือไอเทมที่ต้องการความแม่นยำสูง

## 2. TRun (บอทแปลอัตโนมัติระดับอุตสาหกรรม)
- **หน้าที่:** ใช้โยนไฟล์บทสนทนาขนาดยักษ์ (หลักแสนบรรทัด) ให้ AI แปลรวดเดียว
- **จุดเด่น:** มีระบบเซฟกลางทาง (Checkpoints) เครื่องดับก็กลับมาแปลต่อได้ 
- **เหมาะสำหรับ:** ไฟล์บทสนทนา (Dialogue) ยาวๆ ของตัวละครในเกม

## 3. TPUA (ตัวเข้ารหัสลับสระไทย)
- **หน้าที่:** จัดการสลับรหัสคำไทยปกติ ให้กลายเป็นรหัส PUA (Private Use Area) เพื่อนำไปป้อนเข้าตัวเกม
- **จุดเด่น:** มาพร้อมกับปุ่มเช็กความพังของสระและวรรณยุกต์ซ้อน (Hallucination Checker)

## 4. TGlyph (เครื่องจักรผลิตรูปภาพฟอนต์)
- **หน้าที่:** นำรหัส PUA ที่ได้ ไปวาดรูปภาพตัวอักษรให้กลายเป็นไฟล์ภาพ Font Atlas (รองรับทั้งแบบภาพปกติและ SDF)
- **จุดเด่น:** คำนวณพิกัด (X,Y) ให้อัตโนมัติ พร้อมส่งออกไปยัดใส่ก้อนไฟล์ `.pak` ของเกมได้ทันที

## 5. THub Workspace (การจัดโครงสร้างโฟลเดอร์ทำงาน)
เมื่อกดสร้างโปรเจกต์ใหม่จากหน้า Dashboard ตัวโปรแกรมจะสร้างโฟลเดอร์มาตรฐานให้ 6 โฟลเดอร์ พร้อมไฟล์จัดการอีก 2 ไฟล์ดังนี้ เพื่อควบคุมการผลิตม็อดได้อย่างมืออาชีพ:
* **`01_Original_Backup`**: ใช้เก็บของดั้งเดิมภาษาอังกฤษจากตัวเกม ปลอดภัย ไม่โดนเขียนทับ
* **`02_Translation_Workspace`**: โฟลเดอร์สำหรับนำตารางแปล (.csv, .json) มาทำงานกับ TStudio
* **`03_Font_and_UI`**: ใช้สร้างฟอนต์ แผ่น Texture พิกัดฟอนต์ และเก็บตาราง PUA
* **`04_Packed_Mod`**: โฟลเดอร์ประกอบโครงสร้างตัวม็อดให้เหมือนไฟล์ของเกมจริงเพื่อรันเทสต์
* **`05_Scripts_and_Tools`**: เก็บสคริปต์ Python หรือโปรแกรมแพ็ก/แตกไฟล์เสริมเฉพาะเกมนั้น ๆ
* **`06_Releases`**: ที่เก็บไฟล์ม็อดสำเร็จรูปสุดท้าย (.zip, .rar) สำหรับส่งต่อให้คนอื่นทดลองหรือแจกจ่าย
* **`thub_project.json`**: ไฟล์เซ็ตติ้งเก็บข้อมูลโปรเจกต์ เช่น พาธตัวเกม เพื่อให้โปรแกรม TRun ดึงไปใช้งานรันเกมออโต้
* **`README.md`**: ไฟล์คำแนะนำสำหรับจดบันทึกย่อหรือวิธีการพัฒนาเพิ่มเติม

> [!IMPORTANT]
> หลังจากที่คุณใช้ Extractor แงะไฟล์ข้อความออกจากก้อน `.pak` ของเกมได้แล้ว ให้นำไฟล์มาแบ็คอัปไว้ที่ `01_Original_Backup` ➡️ นำไฟล์แปลมาทํางานใน `02_Translation_Workspace` ผ่าน `TStudio` ➡️ แก้ไขสระลอย/สร้างภาพฟอนต์ที่ `03_Font_and_UI` ➡️ ประกอบโฟลเดอร์ทดสอบที่ `04_Packed_Mod` เพื่อใช้ TRun เปิดเทสต์ ➡️ เขียนสคริปต์บิลด์และเก็บโปรแกรมแปลงไฟล์ไว้ใน `05_Scripts_and_Tools` ➡️ และแพ็กแจกจ่ายที่ `06_Releases`!
