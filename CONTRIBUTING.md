# 🤝 ยินดีต้อนรับสู่การร่วมพัฒนา THub Launcher (Contributing Guide)

ก่อนอื่นเลย พวกเราขอขอบคุณที่คุณให้ความสนใจในการเข้ามาเป็นส่วนหนึ่งของ **THub Launcher**! 🎉
โปรเจกต์นี้ตั้งใจสร้างขึ้นมาเพื่อให้ชุมชนคนทำม็อดเกมและนักแปลได้มีสภาพแวดล้อมและเครื่องมือที่ดีที่สุดใช้งานร่วมกัน และแน่นอนว่าพลังของชุมชน (Community) คือสิ่งที่จะขับเคลื่อนให้โปรเจกต์นี้เติบโตไปได้ไกลที่สุดครับ

คุณไม่จำเป็นต้องเขียนโค้ดเก่งก็สามารถช่วยพัฒนาโปรเจกต์นี้ได้! นี่คือวิธีต่างๆ ที่คุณสามารถเข้ามาร่วมสนุกกับเรา:

---

## 💬 1. การแสดงความคิดเห็น แจ้งบั๊ก หรือเสนอไอเดียใหม่ (Issues)

หากคุณเจอบั๊ก, โปรแกรมค้าง, หรือมีไอเดียว่า "THub น่าจะมีฟีเจอร์นี้เพิ่มนะ" คุณสามารถบอกเราได้เลย!

- **วิธีทำ:** ไปที่แท็บ **[Issues](../../issues)** ด้านบนของหน้า GitHub
- คลิกปุ่ม `New Issue`
- เลือกหมวดหมู่ (เช่น 🐛 Bug Report หรือ ✨ Feature Request)
- เขียนอธิบายรายละเอียดสิ่งที่คุณเจอให้ชัดเจนที่สุด (ถ้ามีรูปภาพประกอบหรือสเปคคอมพิวเตอร์ด้วย จะยอดเยี่ยมมากครับ!)

---

## 🛠️ 2. การช่วยเขียนโค้ด (Pull Requests)

สำหรับนักพัฒนา Python หรือสายเขียนโค้ดที่อยากช่วยแก้บั๊ก หรือเพิ่มฟีเจอร์เข้าสู่โปรแกรมโดยตรง!

### กติกาเบื้องต้น:
1. **Fork** โปรเจกต์นี้ไปยัง GitHub ของคุณเอง
2. **Clone** โปรเจกต์จากเครื่องของคุณมาที่คอมพิวเตอร์
3. สร้าง Branch ใหม่สำหรับฟีเจอร์ที่คุณจะทำ (เช่น `git checkout -b feature/add-new-theme`)
4. เขียนโค้ดของคุณ (THub เขียนด้วย Python และ `customtkinter`)
5. ทดสอบรันโค้ดว่าทำงานได้ปกติ ไม่มี Error
6. **Commit** สิ่งที่คุณแก้ (เช่น `git commit -m "Add new Dracula theme"`)
7. **Push** ขึ้นไปยัง GitHub ของคุณ
8. กดปุ่ม **Create Pull Request (PR)** มายังโปรเจกต์หลักของเรา!

*(หากคุณตั้งใจจะเพิ่มระบบใหญ่ๆ แนะนำให้เปิด Issue มาพูดคุยกับทีมพัฒนาก่อนเริ่มเขียนโค้ด เพื่อดูว่าตรงกับทิศทางของโปรเจกต์หรือไม่ครับ)*

---

## 📖 3. การช่วยเพิ่ม "Modding Knowledge" (คลังความรู้)

THub Launcher มีหมวดหมู่ **Knowledge Base** ซึ่งเป็นไฟล์ Markdown (`.md`) ในโฟลเดอร์ `/Modding-Knowledge` 
หากคุณมีเทคนิคการม็อดเกม, วิธีแกะฟอนต์เอนจิ้นแปลกๆ, หรือสูตรลับในการแปลภาษา คุณสามารถเขียนบทความ Markdown แล้วส่ง Pull Request เข้ามาเพิ่มในโฟลเดอร์นี้ได้เลย! ชุมชนนักม็อดจะได้เรียนรู้จากคุณครับ

---

## 🧩 4. แนะนำโปรแกรมเข้า "Tool Library" และ "Flagship"

หากคุณมีโปรแกรมเจ๋งๆ หรือ Mini-tool เด็ดๆ ที่อยากให้ไปโผล่ในหน้าต่างโหลดโปรแกรมของ THub คุณสามารถเข้ามาแนะนำได้ที่ช่องทาง **Issues** หรือแก้ไขไฟล์ `flagship_list.json` / โค้ดส่วน `TOOL_REGISTRY` และส่ง PR เข้ามาได้เลย!
