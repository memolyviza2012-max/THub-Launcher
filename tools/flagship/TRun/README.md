# 🏃‍♂️ TRun (Batch Game Testing Utility) - คู่มือการใช้งานฉบับสมบูรณ์

TRun เป็นเครื่องมือสาย Automation สำหรับนักม็อดที่ขี้เกียจก๊อปปี้ไฟล์ไปๆ มาๆ ระหว่างโฟลเดอร์ทำงานกับโฟลเดอร์เกม ช่วยประหยัดเวลาในการเทสต์เกมได้มหาศาล

## 🌟 ฟีเจอร์ทั้งหมดแบบเจาะลึก
1. **One-Click Execution:** แทนที่จะต้องก๊อปไฟล์ม็อด > ไปที่โฟลเดอร์เกม > วางทับ > เปิด Steam > กดรันเกม... TRun ยุบทุกอย่างเหลือแค่การ "กดปุ่มเดียว"
2. **Engine Presets:** รองรับโครงสร้างการวางไฟล์ม็อดของเอนจิ้นดังๆ:
   - **Unreal Engine:** รองรับการแพ็ก `.pak` และวางใน `~mods`
   - **Unity Engine:** รองรับการสวอปไฟล์ `resources.assets` หรือ `sharedassets0.assets`
3. **Safe Auto-Backup:** ก่อนที่โปรแกรมจะเอาไฟล์ม็อดของคุณไปวางทับ มันจะแบ็คอัปไฟล์ต้นฉบับของเกมให้เป็น `_original` เสมอ เพื่อความปลอดภัย
4. **Live Log Monitor:** หน้าต่างคอนโซลจำลองที่รายงานสถานะว่าตอนนี้ก๊อปปี้ไฟล์ไหนไปบ้าง และเกมตอบสนองอย่างไร

## 🚀 วิธีใช้งานทีละขั้นตอน (Step-by-Step)
1. **ตั้งค่าโฟลเดอร์เกม:** เปิด TRun ขึ้นมา ในช่อง `Game Executable Path` ให้ Browse ไปหาไฟล์ `.exe` ของเกมที่คุณจะเทสต์
2. **ตั้งค่าโฟลเดอร์ม็อด:** ในช่อง `Mod Build Folder` ให้ Browse ไปหาโฟลเดอร์ที่คุณแพ็กไฟล์แปลไทยเอาไว้
3. **เลือก Engine:** เลือกรูปแบบโครงสร้างของเกม (เช่น Unreal Engine 4/5)
4. **เทสต์เกม:** กดปุ่ม **🚀 RUN TEST**! โปรแกรมจะจัดการก๊อปปี้ไฟล์ แบ็คอัป และเปิดเกมให้คุณทันที
5. **การกู้คืน (Restore):** หากม็อดทำเกมแครช ให้กดปุ่ม `Restore Original` โปรแกรมจะดึงไฟล์แบ็คอัปกลับมาให้เกมกลับสู่สภาพเดิม
