# 🎬 TVox (FMV Subtitle & Video Player) - คู่มือการใช้งานฉบับสมบูรณ์

TVox คือเครื่องมือขั้นเทพที่สร้างมาเพื่อคอเกมสายเนื้อเรื่อง (FMV / Cinematic) ช่วยให้คุณกะจังหวะซับไตเติ้ลไทยให้ตรงกับปากตัวละครได้อย่างแม่นยำระดับมิลลิวินาที

## 🌟 ฟีเจอร์ทั้งหมดแบบเจาะลึก
1. **Native Video Engine:** รองรับการเล่นวิดีโอเกมหลากนามสกุล (mp4, mkv, avi และไฟล์ที่ผ่าน codec เกมอย่าง usm, bk2 หากติดตั้งตัวแปลง) เล่นลื่นไหลไม่มีกระตุก
2. **Interactive Timeline:** ไทม์ไลน์ซับไตเติ้ลแบบลากวางได้ (Drag & Drop) คุณสามารถยืด-หด ความยาวของซับไตเติ้ลบนหน้าจอวิดีโอได้โดยตรง
3. **Live Overlay Render:** ดูผลลัพธ์การเรนเดอร์ซับไตเติ้ลภาษาไทยทับซ้อนลงบนวิดีโอแบบสดๆ พร้อมปรับแต่งขอบฟอนต์ (Outline/Shadow) ให้เหมือนในเกมจริง
4. **Dual Subtitle Track:** เปิดซับต้นฉบับ (อังกฤษ) และซับแปล (ไทย) ขนานกันได้ เพื่อการอ้างอิงที่แม่นยำ
5. **Audio Waveform View:** แสดงกราฟคลื่นเสียง เพื่อให้คุณสามารถกะจังหวะที่ตัวละครเริ่มพูดและหยุดพูดได้เป๊ะ 100%

## 🚀 วิธีใช้งานทีละขั้นตอน (Step-by-Step)
1. **เตรียมไฟล์วิดีโอ:** แตกไฟล์วิดีโอคัตซีนออกจากเกม (เช่นไฟล์ฉากจบ)
2. **นำเข้าวิดีโอและซับไตเติ้ล:** ลากไฟล์วิดีโอใส่หน้าต่างพรีวิว และลากไฟล์ `.srt` หรือ `.vtt` ใส่ลงในช่อง Timeline
3. **เริ่มแปลและจับเวลา:**
   - กดปุ่ม `Spacebar` เพื่อเล่น/หยุดวิดีโอ
   - เมื่อถึงจุดที่ตัวละครพูด ให้ดับเบิ้ลคลิกบล็อกซับไตเติ้ลบนไทม์ไลน์เพื่อพิมพ์คำแปลภาษาไทย
   - ใช้เมาส์ลากขอบซ้าย/ขวาของบล็อก เพื่อขยับเวลาปรากฏ-หายของซับ
4. **ปรับแต่งการแสดงผล:** เลือกฟอนต์และขนาดซับให้เหมือนกับเอนจิ้นเกม เพื่อดูพรีวิวให้เนียนที่สุด
5. **บันทึก:** กดปุ่ม **Export Subtitle** โปรแกรมจะเซฟไฟล์ซับไตเติ้ลใหม่ที่มีการแก้เวลา (Timecode) และคำแปลไทย นำไปแพ็กเข้าเกมได้ทันที
