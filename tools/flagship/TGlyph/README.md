# 🆎 TGlyph (Font Texture Generator) - คู่มือการใช้งานฉบับสมบูรณ์

TGlyph คือเครื่องปฏิกรณ์นิวเคลียร์ในการสร้างฟอนต์! มันจะทำหน้าที่แปลงฟอนต์ `.ttf` ธรรมดาๆ ให้กลายเป็นแผ่นภาพ Texture (Sprite Sheet) พร้อมคำนวณพิกัด Kerning เพื่อใช้ในเกม

## 🌟 ฟีเจอร์ทั้งหมดแบบเจาะลึก
1. **Atlas Packer (`atlas_packer.py`):** คำนวณการแพ็กตัวอักษรลงในสี่เหลี่ยมจัตุรัสให้ประหยัดพื้นที่ที่สุด
2. **SDF Font Renderer (`font_renderer.py`):** สร้างฟอนต์แบบ Signed Distance Field (SDF) ซึ่งเป็นเทคโนโลยีที่ทำให้ฟอนต์ในเกม 3D ขยายใหญ่แค่ไหนก็ไม่แตก!
3. **DDS Converter (`dds_converter.py`):** มีตัวแปลงไฟล์ให้เป็นนามสกุล `.dds` (DirectDraw Surface) รองรับการบีบอัดแบบ DXT/BC รันเข้าการ์ดจอโดยตรง ช่วยลดการกินแรมของเกม
4. **Kerning & Text Patcher (`text_patcher.py`):** ดึงค่า Kerning (ระยะห่างระหว่างตัวอักษร) ออกมาบันทึกเป็นไฟล์พิกัด `.xml`, `.fnt`, หรือ `.json` เพื่อให้เอนจิ้นเกมรู้ว่าต้องตัดภาพตรงไหน

## 🚀 วิธีใช้งานทีละขั้นตอน (Step-by-Step)
1. **โหลดฟอนต์ต้นฉบับ:** เปิดโปรแกรม กดปุ่มเลือกไฟล์ Font (`.ttf` หรือ `.otf`)
2. **ตั้งค่าความละเอียด:** เลือกขนาด Base Font Size (แนะนำ 64 หรือ 128 สำหรับเกม HD)
3. **ใส่ตัวอักษรเป้าหมาย:** วางชุดอักขระ (Character Set) ภาษาไทย ก-ฮ สระ วรรณยุกต์ (รวมถึง PUA ถ้ามี) ลงในช่อง
4. **เลือก Output:** ติ๊กเลือกรูปแบบไฟล์ที่เกมต้องการ (เช่น .png, .dds, .fnt, .xml)
5. **สร้างภาพฟอนต์:** กด **Generate Texture Atlas** โปรแกรมจะใช้เวลาวาดภาพสักครู่
6. คุณจะได้โฟลเดอร์ Output ที่ประกอบไปด้วยแผ่นภาพฟอนต์และไฟล์พิกัด นำไปใช้ในเกมได้เลย!
