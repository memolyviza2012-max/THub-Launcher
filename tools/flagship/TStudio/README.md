# 📝 TStudio (AI Translation Editor) - คู่มือการใช้งานฉบับสมบูรณ์

TStudio เป็นสุดยอดโปรแกรมสำหรับนักม็อดแปลภาษาเกม ที่มาพร้อมฟีเจอร์ระดับมืออาชีพ ผสานพลัง AI เพื่อให้งานแปลของคุณง่าย เร็ว และมีคุณภาพสูงสุด

## 🌟 ฟีเจอร์ทั้งหมดแบบเจาะลึก
1. **Multi-AI Engine Support:** รองรับ API จาก AI ชั้นนำระดับโลก ทั้ง Google Gemini, Anthropic Claude, OpenAI, DeepSeek รวมถึง Local LLM (รัน AI ในเครื่องตัวเอง)
2. **Smart Format Converter:** รองรับไฟล์เกมหลากหลายนามสกุล (`.csv`, `.lor`, `.locres`, `.tex`) และมีระบบตารางแมปปิ้งคอลัมน์อัตโนมัติเมื่อโหลดไฟล์แปลกๆ
3. **Advanced Translation Modes:**
   - **🤖 Re-translate (AI):** แปลทีละบรรทัดแบบมาตรฐาน
   - **🎯 Re-translate (3 Options):** ให้ AI เสนอคำแปลมา 3 สไตล์พร้อมกัน (ตรงตัว, ภาษาพูด, ทางการ) ให้คุณเลือกจิ้มได้เลย
   - **✨ แปลพิเศษ (Special Modes):** โหมดแปลกวีนิพนธ์ (Poem), แปลคำคม (Quote), และแปลสำนวน (Idiom) ให้เข้ากับบริบทเกม
4. **Glossary & Prompt Injection:** ระบบพจนานุกรมคำศัพท์ (Glossary) ที่คุณตั้งชื่อเฉพาะ (เช่น ชื่อเมือง, ชื่อตัวละคร) แล้วระบบจะฝังคำพวกนี้เข้าไปสั่ง AI อัตโนมัติ AI จะแปลชื่อไม่ผิดเพี้ยน!
5. **Smart Filtering:** ค้นหาและกรองข้อมูลขั้นเทพ เช่น "กรองเฉพาะบรรทัดที่ยังไม่แปล", "กรองบรรทัดที่ AI อาจจะหลอน (Hallucination check)", และ "กรองหาประโยคคำคม"
6. **Direct Deployment:** ปุ่ม `🧩 Export PUA` สำหรับส่งไฟล์ไปทำฟอนต์ และ `🚀 Deploy to Game` ยิงไฟล์เข้าเกมทันที

## ⚙️ วิธีการตั้งค่าก่อนเริ่มใช้งาน
1. เปิดโปรแกรม TStudio
2. ไปที่ปุ่ม **⚙️ Global Settings** ด้านบนขวา
3. นำ API Key ของ AI ที่คุณต้องการใช้ (เช่น Gemini หรือ DeepSeek) มาใส่ในช่อง แล้วกด Save
4. หากคุณมีชื่อตัวละครที่ต้องการให้แปลตรงกันตลอด ให้กดปุ่ม **📖 Glossary** แล้วเพิ่มคำศัพท์ลงไป

## 🚀 วิธีใช้งานทีละขั้นตอน (Step-by-Step)
1. **นำเข้าไฟล์:** กด `📂 Open File` แล้วเลือกไฟล์ข้อความของเกม (เช่น `translation.csv`)
2. **การจัดการตาราง:** หน้าจอจะแบ่งเป็นฝั่งซ้าย (ภาษาอังกฤษต้นฉบับ) ฝั่งขวา (คำแปลไทย) และตรงกลางเป็นคำแปลอ้างอิงจาก AI
3. **การแปล:** คลิกที่บรรทัดใดก็ได้ กดปุ่ม `🤖 Re-translate` ระบบจะดึงคำแปลมาให้ในเสี้ยววินาที
4. **ตรวจสอบและแก้ไข:** คุณสามารถพิมพ์แก้คำแปลฝั่งขวาได้ตามใจชอบ
5. **แก้ปัญหาฟอนต์ลอย:** หากเกมมีปัญหาสระลอย ให้กดปุ่ม `🧩 Export PUA` เพื่อส่งไฟล์นี้ไปให้โปรแกรม TPUA จัดการ
6. **บันทึกผล:** กด `💾 Save CSV` หรือ `Ctrl+S` เพื่อบันทึกงาน
