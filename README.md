# 🚀 THub Launcher

**THub Launcher** คือศูนย์รวมและจุดศูนย์กลาง (Hub) สำหรับนักม็อดเกมและผู้ใช้งานทั่วไป! 
เป้าหมายหลักของ THub คือการเป็นศูนย์รวมเครื่องมือ (Modding Tools) ทั้งหมดให้อยู่ในที่เดียว โดยมีระบบการจัดการที่ใช้งานง่าย มี UI ที่สวยงาม และมีระบบ Cloud Registry ที่ช่วยให้ผู้ใช้งานได้รับเครื่องมือเวอร์ชันใหม่ล่าสุดอัตโนมัติ

## 🌟 ฟีเจอร์หลัก (Key Features)

- **💎 Flagship Products:** ดาวน์โหลดและติดตั้งสุดยอดเครื่องมือ (T-Series) ทั้ง 5 ตัว (TStudio, TRun, TPUA, TGlyph, TVox) ได้โดยตรงเพียงคลิกเดียว ระบบจะติดตั้งและเตรียมพร้อมใช้งานให้ทันที
- **☁️ Cloud Registry:** สมุดหน้าเหลืองออนไลน์ที่เชื่อมต่อกับ GitHub ทำให้สามารถดึงรายการโปรแกรมและอัปเดตเวอร์ชันใหม่ๆ ได้แบบเรียลไทม์โดยไม่ต้องดาวน์โหลดตัว Launcher ใหม่
- **🧩 Tool Library:** ศูนย์รวมเครื่องมือเสริม (Mini-tools) กว่าสิบชนิดที่คุณสามารถเลือกดาวน์โหลดได้ตามใจชอบ
- **📚 Modding Knowledge:** คลังความรู้และคู่มือการสร้างม็อดเกมแบบเจาะลึก (Engine Hacking, Font Modding) ที่ฝังมาในโปรแกรม
- **🎨 Modern UI:** ดีไซน์หน้าต่างโปรแกรมที่ล้ำสมัย สวยงามสไตล์ Glassmorphism และใช้งานลื่นไหล

## 📥 วิธีติดตั้งและเข้าใช้งาน (Installation)

1. ไปที่หมวด [Releases](https://github.com/memolyviza2012-max/THub-Launcher/releases) ของ Repository นี้
2. ดาวน์โหลดไฟล์ `THub.zip` เวอร์ชันล่าสุด
3. แตกไฟล์ (Extract) ไว้ในโฟลเดอร์ที่คุณต้องการในเครื่อง PC ของคุณ
4. ดับเบิ้ลคลิกที่ไฟล์ **`THub.exe`** เพื่อเริ่มต้นเข้าสู่โลกแห่งการม็อดได้ทันที!

## ⚙️ โครงสร้างและการทำงาน (For Developers)
โปรเจกต์นี้เขียนด้วย Python (CustomTkinter) และสถาปัตยกรรมการอัปเดตแบบ Modular:
- ตัวโปรแกรมหลักจะไปอ่าน `flagship_list.json` จาก GitHub
- เมื่อสั่งดาวน์โหลด มันจะไปดึง Source Code (zip) จากหน้า Release ของ Repository ของโปรแกรมนั้นๆ
- ระบบจะทำการแตกไฟล์และรันสคริปต์ / ไฟล์ `.exe` ให้เบื้องหลัง ทำให้ประสบการณ์ผู้ใช้ (UX) ลื่นไหลอย่างไร้รอยต่อ
